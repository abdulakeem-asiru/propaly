## Overview
Propaly was built to eliminate the friction real estate businesses face when setting up their online presence.
Instead of relying on multiple tools for websites, listings, and lead management, Propaly brings everything into a single, streamlined platform.
The solution allows property businesses to publish listings, capture and manage leads, and automate follow-upsâ€”helping them move from visibility to real revenue with less effort.

## Key Features
- Instant Website & Listing Setup
  Launch a fully functional real estate website with structured property listings in minutes.

- Property Management System
  Easily create, update, and manage property listings with a clean and scalable data model.

- Lead Capture & Management
  Built-in lead forms connected directly to listings, ensuring no inquiry is lost.

- Automated Marketing Workflows
  Automatically nurture leads through follow-up flows to increase inquiries and conversions.

- Conversion-Focused UI
  Designed with clarity and user flow in mind to guide visitors from browsing to inquiry.

## Tech Stack
- Frontend: React, Next.js, Tailwind CSS
- Backend: NodeJS
- State & Data: React Query & Zustand
- Animations: GSAP
